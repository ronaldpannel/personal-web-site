There are many brilliant people who have unknowingly helped me on my coding journey. in the cards below I have referenced and acknowledged the people and learning resources I access and use on a regular basis.

My favorite coding youtube channels

Frank's Laboratory

Frank's YouTube channel

Frank teaches Javascript by building art animations and games in Html Canvas without the use of any Javascript libraries.

Franks tutorials always start beginner friendly and then build in complicity to include intermediate and advanced concepts.

Franks uses OOP in many of his projects, it was by watching Franks tutorials that gave me a understanding of constructor functions, ES6 classes, inheritance, polymorphism, etc. Frank also introduces more advanced Html markup, this channel is not suitable for the absolute beginner, you need to know basic javascript fundamentals to benefit from Franks tutorials.

Code with Ania Kubów

Ania's YouTube channel

Ania teaches Javascript by coding games in the DOM. no Html canvas. This channel also includes a free 12hr javascript Bootcamp

With this Bootcamp Ania does a brilliant job of condensing all the key javascript topics down to the minimum no extra unnecessary information. On completing Aina's Bootcamp I compiled all the code into a word document which as given me a great reference document which I continually refer too.

Ania's approach to teaching javascript by coding games in the DOM demonstrates all the principles of manipulating the DOM. by watching and coding along with Ania it helped me see all the principles I had learned in the Udemy courses applied and combined to achieve a desired end result.

The breath of Ania's knowledge based on the wide range of coding tutorial subjects on her channel is phenomenal.

The Net Ninja with Shaun Pelling

Shaun's YouTube channel

The Net Ninja channel is a great resource for learning both front and back end web development from beginner to advanced.

Shaun's youtube channel contains a series of modern javascript tutorial playlists for all skill levels.

This channel also includes tutorials on CSS, React, React Native, MongoDB, Docker, Vue, Bootstrap, and so much more .

Shaun also has a beginner to advanced Javascript course on Udemy which I highly recommend.

The Coding Train with Daniel Shiffman

Daniel's YouTube channel

Apart from the massive volume of high quality content, it is Daniel that makes this channel so good, Daniel's enthusiasm for coding and teaching coding is infectious.

In Daniels own words this channel is about-

"All aboard! The Coding Train is on its way with creative coding video tutorials on subjects ranging from the basics of programming languages like JavaScript to algorithmic art, machine learning, simulation, generative poetry, and more. Choo choo!"

Daniel uses P5js to teach javascript, although I believe it is essential to learn javascript to a reasonable skill level before using libraries. However, for creating digital art and visualizations in Html canvas working with vectors I consider P5js an exception.

P5js significantly reduces the amount of code required when working with vectors compared to vanilla javascript. P5js makes applying vector maths simple compared to Javascript which would require creating a dedicated vector class to handle vector math.

Before starting a project Daniel always explains the sometimes complex formulas and Algorithms in as simple terms as possible. Daniel then demonstrates how to breakdown formulas and Algorithms into manageable chunks making converting into code a simpler and more logical process.

Daniel always makes sure the resources researched for any of his tutorials are well referenced.

The Coder Coder with Jessica Chan

Jessica's YouTube channel

This is great resource for beginners wanting to learn front end web design with Html, CSS, Sass and Javascript. Jessica has a brilliant teaching style,

apart from the coding aspect Jessica explains her thought process for achieving an engaging UI design, and how she creates simple notes listing the projects functionality allowing a methodical approach to be applied to compiling the code.

In some of the tutorials Jessica selects elements that she may not fully understand how to code allowing her to demonstrate how she researches topics, which is a skill in it's self.

This channel is very good for learning Sass, Jessica uses it in most of her tutorials and explains the subject in great detail.

Kevin Powell

Keven's's YouTube channel

Kevin is a CSS master, hence, this is the best channel i have found for learning CSS from beginner to advanced.

Kevin's tutorials are generally short and condensed, where he will pick a specific CSS topic and teach the subject in great detail.

Most of Kevin's tutorials focus the CSS required for creating responsive and dynamic web sites.

Kevin uses Sass in a lot of his tutorials and explains it and the naming conventions in great detail.

Online Tutorials with Muhammad Irshad

Muhammad's YouTube channel

The CSS animations on this channel are at another level, there is no dialog in the tutorials the coding is displayed with music playing in the background,

therefore, not really suitable for the absolute beginner

This channel contains over 1400 videos, the channel is well organized so finding what you want is not normally a problem. The channel covers all aspects of front end web development.

I only use this channel for advanced CSS animation learning and inspiration.

In Muhammad's own words "Online Tutorials is a channel for learning Complete responsive website design, Creative CSS animation and Hover Effects,Amazing Vanilla Javascript projects, creative UX and UI Design using Html and CSS and much more...
